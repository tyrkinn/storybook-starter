# Storybook UI kit

This starter provides a minimal setup to get Storybook working with typescript and Vite.


# Features

- [x] Path Aliases
- [x] Eslint Config
- [x] StoryBook

# Using starter

```bash
$ npx degit tyrkinn/storybook-starter project-name
```

Than fix the `name` field in `package.json`

# Running App

## Dev mode

```bash
npm run dev
```

## Build

```bash
npm run build
```

# Running Storybook

```bash
$ npm run storybook
```
