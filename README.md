# Storybook UI kit

This starter provides a minimal setup to get Storybook working with typescript and Vite.


# Features

- [x] Path Aliases
- [x] Eslint Config
- [x] StoryBook

# Running App

## Dev mode

```bash
npm run dev
```

## Build

```bash
npm run build
```

# Running Storybook

```bash
$ npm run storybook
```
